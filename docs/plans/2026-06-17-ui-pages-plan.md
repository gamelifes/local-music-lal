# UI 页面开发计划（按照原型还原）

## 目标

按照 `MusicFree_A_final.html` 原型的视觉设计和交互逻辑，开发所有 UI 页面。

## 从原型提取的设计规范

### 颜色系统
```css
--accent: oklch(0.72 0.16 82);        /* 琥珀金主色 */
--accent-glow: oklch(0.72 0.16 82 / 0.4);  /* 琥珀金发光 */
--bg: #0F0F0A;                         /* 深色背景 */
--bg-card: rgba(255,255,255,0.03);     /* 卡片背景 */
--bg-card-hover: rgba(255,255,255,0.06); /* 卡片悬停 */
--glass-border: rgba(255,255,255,0.12); /* 玻璃边框 */
--text: rgba(255,255,255,0.92);        /* 主文字 */
--text-secondary: rgba(255,255,255,0.6); /* 次要文字 */
--divider: rgba(255,255,255,0.12);     /* 分割线 */
```

### 组件样式
- **手机框架**: 375x812px, 圆角 40px, 边框 6px solid #1a1a1a
- **刘海**: 150x28px, 黑色, 底部圆角 16px
- **按钮**: padding 12px 20px, 圆角 12px, 边框 1px solid glass-border
- **卡片**: 圆角 12px, 背景 bg-card
- **输入框**: 圆角 12px, 边框 1px solid glass-border

---

## 开发任务

### Task 1: 创建设计系统组件

**目标**: 创建可复用的 UI 组件库

**组件清单**:
1. `Button` - 按钮组件（primary, danger, default）
2. `Input` - 输入框组件
3. `Card` - 卡片容器
4. `Modal` - 模态框组件
5. `Badge` - 标签组件（音质标签）
6. `Avatar` - 头像组件

---

### Task 2: 创建手机框架

**目标**: 实现手机模拟器布局

**组件**:
- `PhoneFrame` - 手机外壳（圆角、边框、刘海）
- `StatusBar` - 状态栏（时间、信号）
- `PhoneContent` - 内容区域

**样式要点**:
- 宽度: 375px
- 高度: 812px
- 圆角: 40px
- 边框: 6px solid #1a1a1a
- 刘海: 居中, 150x28px, 黑色

---

### Task 3: 创建导航组件

**目标**: 实现页面导航系统

**组件**:
- `Drawer` - 侧边抽屉导航
- `MusicBar` - 底部迷你播放栏
- `NavBar` - 底部导航栏（可选）

**抽屉导航要点**:
- 宽度: 260px
- 背景: rgba(18,18,16,0.95) + backdrop-filter:blur(24px)
- 滑入动画: transform 0.35s
- 导航项: 首页、歌手、专辑、已隐藏、搜索、均衡器、扫描

**底部播放栏要点**:
- 高度: 72px
- 背景: rgba(255,255,255,0.06) + backdrop-filter:blur(32px)
- 内容: 封面缩略图 + 歌曲信息 + 播放控制

---

### Task 4: 创建首页

**目标**: 实现歌曲列表页面

**组件**:
- `HomePage` - 首页容器
- `SongTable` - 歌曲表格
- `SearchBar` - 搜索栏

**布局要点**:
- sticky 表头（标题 + 表头）
- 歌曲列表可滚动
- 底部留出 MusicBar 空间（88px）

**歌曲表格要点**:
- 表头: 封面(64px) | 歌名 | 歌手 | 时长(60px)
- 封面: 48x48px 圆形, 琥珀金渐变
- 当前播放歌曲高亮（琥珀色文字 + 发光效果）

---

### Task 5: 创建播放页

**目标**: 实现播放器页面

**组件**:
- `PlayerPage` - 播放页容器
- `VinylRecord` - 黑胶唱片组件
- `ProgressBar` - 进度条
- `PlaybackControls` - 播放控制

**布局要点**:
- 背景: #8A8580 (中灰色)
- 唱片区域: flex:1, 居中
- 操作按钮: 音质、分享、睡眠、隐藏
- 进度条: 可点击/拖动
- 控制按钮: 循环、上一首、播放/暂停、下一首、列表

**黑胶唱片要点**:
- 尺寸: 240x240px
- 外圈: conic-gradient 旋转纹理
- 唱片本体: 径向渐变（深黑色 #080808 → #000）
- 封面: 50% 大小, 琥珀金渐变
- 沟纹: repeating-radial-gradient, 透明度 0.15
- 中心标签: 40x40px, 深色半透明
- 旋转动画: 20s 一圈
- 播放时旋转, 暂停时停止

**唱片 ↔ 歌词切换**:
- 左滑/右滑切换
- 底部圆点指示器
- 过渡动画: 0.35s ease

---

### Task 6: 创建歌词页

**目标**: 实现歌词显示页面

**组件**:
- `LyricsPage` - 歌词页容器
- `LyricLine` - 歌词行
- `KaraokeWord` - 卡拉OK逐字

**歌词样式要点**:
- 字体大小: 18px
- 行高: 1.8
- 当前行: 放大(scale 1.05) + 加粗 + 白色
- 已唱行: 琥珀色
- 未唱行: 灰色半透明

**卡拉OK动画**:
- 每字间隔: 200ms
- 当前字: 白色 + 发光效果
- 已唱字: 琥珀色

---

### Task 7: 创建均衡器页

**目标**: 实现均衡器页面

**组件**:
- `EqualizerPage` - 均衡器页容器
- `PresetButton` - 预设按钮
- `FrequencySlider` - 频率滑块

**布局要点**:
- 预设按钮: flex-wrap 布局
- 5 个频率滑块: 60Hz, 230Hz, 910Hz, 3.6kHz, 14kHz
- 滑块范围: -12dB ~ +12dB

---

### Task 8: 创建扫描页

**目标**: 实现文件扫描页面

**组件**:
- `ScanPage` - 扫描页容器
- `ScanAnimation` - 扫描动画
- `FolderList` - 文件夹列表

**布局要点**:
- 扫描动画: 旋转圆圈
- 进度显示: 已发现歌曲数
- 扫描完成后显示结果

---

### Task 9: 创建模态框

**目标**: 实现所有模态框

**组件**:
- `SleepModal` - 睡眠模式
- `QualityModal` - 音质选择
- `ShareModal` - 分享信息
- `FileInfoModal` - 文件信息

**模态框要点**:
- 背景: rgba(0,0,0,0.7)
- 内容: 圆角 20px, 背景 rgba(22,22,20,0.95)
- 关闭按钮: 右上角 ✕
- 过渡动画: opacity 0.3s, scale 0.3s

---

### Task 10: 整合与测试

**目标**: 整合所有组件, 完整测试

**测试项**:
1. 所有页面可正常切换
2. 抽屉导航正常工作
3. 播放页唱片动画正常
4. 唱片 ↔ 歌词切换正常
5. 所有模态框可正常打开/关闭
6. 底部播放栏正常显示
7. 视觉效果与原型一致

---

3. 交互功能正常
4. 代码提交到 git
